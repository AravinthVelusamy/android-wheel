## Getting Started

 > Add the dependency to your build.gradle.
 
```
dependencies {
 	compile 'com.isanwenyu.android-wheel:wheel:2.1'
}
```
- Maven:

```
<dependency>
  <groupId>com.isanwenyu.android-wheel</groupId>
  <artifactId>wheel</artifactId>
  <version>2.1</version>
  <type>pom</type>
</dependency>
```
